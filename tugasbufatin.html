3. Sebutkan gaya interaksi dalam antarmuka pengguna!
4. Apa yang dimaksud dengan user interface, Jelaskan minimal 1 paragraph!
5. Sebutkan karakteristik user interface! dan jelaskan!
6. Kenapa UI dan UX berjalan berdampingan!

JAWABAN


1.
a. Tujuan produk, yakni apa yang ingin dicapai, dan didapatkan dari penggunaan produk interaktif tersebut.
b. Tipe atau model yang akan dirancang

2. penglihatan, luminans, kontras, kecerahan, pov, dan ketajaman penglihatan

3. Command Line, menu selection, form fill in, Direct manipulation.

4. User Interface atau GUI adalah jenis antarmuka (interface) yang menggunakan metode interaksi pada perangkat elektronik secara grafis, tidak hanya berupa perintah teks antara pengguna dan computer (CLI).

5.
•Tampilan visual menarik.
• Interaksi pick and click.
• Memiliki keterbatasan gaya interaksi.
• Visualisasi Berorientasi objek.
• Memanfaatkan memori pengenalan manusia.
• Dapat menjalankan beberapa fungsi sekaligus.

6. Karena, User Interface atau UI merupakan bagian dari komponen User Experience atau UX yang berupa tampilan visual dari sebuah aplikasi. Dapat diumpamakan bahwa User Interface dan User Experience sebagai otak kiri dan otak kanan dalam pengembangan aplikasi.
